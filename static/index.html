Service de cracking — Demo

Utilise l'API POST /api/v1/crack pour soumettre un hash (mode sync pour tests rapides).

Exemple (curl)

curl -s -X POST http://127.0.0.1:5000/api/v1/crack \
  -H "Content-Type: application/json" \
  -d '{"hash":"5f4dcc3b5aa765d61d8327deb882cf99","algo":"md5","mode":"sync","wordlist":"dico.txt"}'

Fichier wordlist par défaut utilisé : dico.txt
